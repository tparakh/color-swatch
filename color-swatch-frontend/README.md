# Color Swatch FrontEnd App

## Getting started

#### Requirement

This app uses node v12.4.0, yarn v1.22.5

#### Project setup

Install project dependencies:

```
nvm install v12.4.0
nvm use v12.4.0
yarn install

```

#### Run development server

```
npm start
```

#### Access Frontend once backend API is up and running

```
http://localhost:3000/

```
